# GridLayoutManager
<b> DYNAMIC GRIDLAYOUTMANAGER FOR TABLET</b>
<br/><br/><br/>
Easily make two or more columns of recyclerview layout.
```ruby
implementation 'com.github.emrekotun:GridLayoutManager:1.0'
```
